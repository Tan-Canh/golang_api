## Golang connect Postgres
### APIs users demo
```gotemplate
api: /users         ==> add user
api: /users         ==> get list user
api: /users/:id     ==> get user by id
api: /users/:id     ==> update user
```

Clone: ``https://github.com/Tan-Canh/golang_api.git``

Add package: ``go get ./...``

Run: ``go run main.go``
